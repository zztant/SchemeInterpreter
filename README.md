#A simple interpreter for functional language Scheme

## Main implementations:
 1.car, cdr, pair, list, lambda, let ,cond, if, call/cc, define, define-syntax and so on

 2.macro with no hygiene
 
 3.call/cc
 
 4.garbage collector

 1. Ubuntu14.04 with clang-3.8
